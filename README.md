## What was used in this project?
1 - Typescript
2 - Node + Express 
3 - Prisma (ORM)
4 - Jest (Tests[Unit and Integration])
5 - SQLite (Database)

```
# clone the repository
$ git clone git@github.com:NicolasBinatti/textoit-api.git

# install dependences
$ npm install

# run the projetct
$ npm start

#run testes
$ npm test
```
